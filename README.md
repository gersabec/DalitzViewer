# DalitzViewer
Authors: Yinrui Liu (University of the Chinese Academy of Sciences), Marco Gersabeck (The University of Manchester)
Version: 1.0.0

A tool to visualise resonant behaviour in a Dalitz plot, including CP asymmetries.
